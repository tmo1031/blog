{% assign tags = '' | split: '' %} {% for t in site.tags %} {% assign tags = tags | push: t[0] %} {% endfor %} {%- comment -%} Create a new array to hold sortable strings. The string format is "post_count::tag_name". We pad the count with leading zeros to ensure correct string sorting. {%- endcomment -%} {%- assign tag_strings = "" | split: "" -%} {%- for tag in site.tags -%} {%- comment -%} Prepend '0000' and take the last 4 characters for padding {%- endcomment -%} {%- assign post_count = tag[1].size | prepend: "0000" | slice: -4, 4 -%} {%- assign tag_string = post_count | append: "::" | append: tag[0] -%} {%- assign tag_strings = tag_strings | push: tag_string -%} {%- endfor -%} {%- comment -%} Sort the strings descending (Z->A), which works because of the padded count {%- endcomment -%} {%- assign tags = tag_strings | sort | reverse -%} {%- for item in tags -%} {%- assign parts = item | split: "::" -%} {%- assign post_count = parts[0] | plus: 0 -%} {%- comment -%} Convert back to number {%- endcomment -%} {%- assign tag_name = parts[1] -%} {{- tag_name -}} {{ post_count }} {%- endfor -%}
